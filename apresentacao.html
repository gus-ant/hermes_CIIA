O que é o Hermes?

O Hermes é uma plataforma de agentes de IA que executa tarefas reais utilizando ferramentas locais e serviços externos.

Diferente de um chatbot tradicional, cada agente possui uma identidade, um conjunto de ferramentas e uma especialização.

  • Executa comandos no computador quando necessário
  • Consulta documentos e arquivos do Google Drive
  • Navega na Web utilizando navegador automatizado
  • Responde pelo Discord para toda a equipe
  • Mantém memória e contexto de cada agente

Arquitetura do Projeto

Os agentes executam localmente no computador utilizando o Hermes como plataforma de orquestração.

  • Perfis independentes (Jo, Larissa e Vini)
  • Comunicação via Discord
  • Ferramentas locais e APIs externas
  • OpenRouter para acesso aos modelos LLM
  • Possibilidade futura de executar modelos locais

Por que criar agentes diferentes?

O CIIA reúne pessoas de diferentes áreas do conhecimento.

Em vez de utilizar um único assistente genérico, optamos por criar agentes especializados em domínios distintos.

  • Jo: Organização administrativa
  • Larissa: Comunicação e apoio aos bolsistas
  • Vini: Tecnologia e automação

Integração com o Discord

Todos os agentes ficam disponíveis diretamente no servidor do Discord do CIIA.

  • cada agente possui um bot próprio
  • basta mencionar o agente para iniciar uma conversa
  • vários membros podem utilizá-los simultaneamente
  • integração com Google Drive e outras ferramentas
  • execução local preservando maior controle dos dados
