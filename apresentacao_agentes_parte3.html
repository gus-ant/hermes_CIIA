Para preparar o terreno, precisamos baixar apenas duas ferramentas. A primeira é o Antigravity 2.0, um assistente autônomo que age como um desenvolvedor particular, fazendo o trabalho pesado por você. A segunda é o Visual Studio Code, ou VS Code.

Por que o VS Code?

"Mas eu não sei programar! Pra que um editor de código?"

  • Na prática, o VS Code é apenas um Bloco de Notas com superpoderes.
  • Nós vamos usá-lo apenas para abrir um arquivo seguro chamado .env.
  • É lá que nós colaremos a nossa Chave de API, para que fique escondida e segura no seu computador.
Muitas pessoas olham pro VS Code e entram em pânico achando que terão que programar em Matrix. Mas na verdade, para leigos, o VS Code funciona como um Bloco de Notas mais chique e seguro. Nós baixamos ele apenas porque o bloco de notas normal do Windows as vezes estraga arquivos de configuração. Nós vamos abrí-lo, clicar no arquivo .env (que é um cofre do agente), colar um texto lá e fechar. Simples assim.

Perdendo o Medo da Tela Preta

O que é o Terminal no seu Computador?

O terminal nada mais é do que um chat direto com o seu computador. Em vez de usar o mouse para clicar em pastas e ícones, você conversa com o sistema digitando ordens simples.

🪟 No Windows: Ele se chama PowerShell ou Prompt de Comando (CMD). (Basta pesquisar no Iniciar!).
🍎 No Mac / Linux: Ele se chama simplesmente Terminal.

Para que vamos usá-lo hoje?

  • Para autorizar o nosso agente com a chave de API do OpenRouter.
  • Para acordar (ativar) o cérebro do Hermes no seu sistema.
Outro pavor clássico de quem está começando é a famosa 'tela preta'. Mas para desmistificar: o terminal é apenas um chat do WhatsApp, só que em vez de falar com uma pessoa, você fala com o núcleo do seu sistema operacional. Se você usa Windows — como a maioria de nós —, no seu computador ele vai aparecer com o nome de 'PowerShell' ou 'Prompt de Comando'. No Mac, ele se chama apenas 'Terminal'. É a mesma coisa! Nós vamos usar essa janela apenas para duas tarefas simples: salvar as credenciais e apertar o botão de 'Play' no nosso agente.

Como o Antigravity Trabalha?

O que a IA está fazendo nos bastidores da sua máquina:

Terminal

O agente executa comandos no seu terminal para baixar dependências e criar pastas.

Leitura de Arquivos

Ele analisa o seu computador para saber onde organizar o sistema.

Edição de Código

E enquanto o Antigravity trabalha, o que ele está fazendo exatamente? Ele está usando as próprias ferramentas que descrevemos na aula anterior. Ele usa a ferramenta Terminal para clonar repositórios, a de Leitura para ver se a pasta já existe, e a de Edição para preparar os arquivos-base. É uma IA configurando outra IA.

Passo 1: A Chave no OpenRouter

A Alma do Seu Agente

A chave de API é o "passaporte" que conecta o Hermes aos super cérebros como ChatGPT, Llama 3 ou Claude.

  1. Acesse o site openrouter.ai
  2. Crie uma conta gratuita
  3. Vá em "Keys" e clique em Create Key
  4. Copie o código gerado (ex: sk-or-v1-xxxx...)
Assim que a instalação termina, o agente existe, mas é como um cérebro desligado. A inteligência vem da Chave de API, que conecta a estrutura local à capacidade de raciocínio de ponta. Acessem openrouter.ai, criem a chave e deixem no CTRL+C. O OpenRouter é excelente porque nos dá acesso a dezenas de modelos de IA num único lugar, pagando apenas centavos por uso.

Passo 2: Configurando no Terminal

Assistente de Configuração

No terminal, digite o comando de setup guiado:

hermes configure
  • Provider: Escolha OpenRouter
  • Model: Escolha claude-3.5-sonnet
  • API Key: Cole sua chave copiada
[INSTRUÇÃO PARA O PROFESSOR: Mostrar o slide e comentar a imagem do print] Como vocês podem ver no print da tela preta, quando digitamos 'hermes configure', o programa nos guia. Ele pergunta: Qual provedor? Escolhemos OpenRouter. Qual modelo? Escolhemos Llama ou Claude. Colar sua chave? Colamos a chave e pronto, o terminal nos diz que a configuração foi salva!

Passo 3: Colando a Chave no VS Code

Validando no VS Code

Caso prefira conferir manualmente, abra a pasta do projeto no VS Code e acesse o arquivo .env.

  • Confera se a chave está em OPENROUTER_API_KEY.
  • Este arquivo é 100% local e seguro no seu computador.
  • O cérebro da IA estará fisicamente conectado!
[INSTRUÇÃO PARA O PROFESSOR: Mostrar o slide e comentar a imagem do VS Code] Para quem preferir, ou para quem precisar validar, é aqui que o Visual Studio Code entra. Olhem para este print: nós abrimos a pasta do projeto no VS Code, clicamos no arquivo .env e vemos nossa chave lá dentro. Ninguém de fora tem acesso a esse arquivo, ele é 100% local. O cérebro está fisicamente conectado à estrutura!

Passo 4: O Despertar do Hermes

Ligando o Agente

No terminal, digite o comando para iniciar a execução:

hermes start

O Gateway conectará à API e carregará o banco de memória SQLite local.

"Vamos fazer uma pergunta para o nosso Hermes?"

E, finalmente, nós voltamos para o terminal e rodamos 'hermes start'. O que vemos agora? [Mostrar Print] Serviços iniciando... Conexão com OpenRouter OK. Memória OK. Hermes Gateway Online. Ele está vivo, rodando silenciosamente no fundo do nosso computador. E para provar que funciona, nós vamos agora, ao vivo, testar essa comunicação: "Vamos fazer uma pergunta para o nosso Hermes?".
