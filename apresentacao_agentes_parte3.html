Módulo 3: O Setup

Instalando e Dando Vida ao Seu Agente

Chegamos ao Módulo 3. Agora que entendemos como o agente funciona na teoria, nós vamos literalmente dar vida a ele no seu computador. Muitas pessoas acham que rodar Inteligência Artificial localmente exige supercomputadores da NASA ou ser hacker de filmes, mas vou mostrar que é muito mais simples. Para essa aula, preparei um passo a passo para leigos, mostrando que a tecnologia está ao alcance de todos. E nós precisaremos apenas de dois programas iniciais.

O Nosso Cinto de Utilidades

O que precisamos baixar antes de começar?

Antigravity 2.0

O assistente de inteligência artificial que fará a instalação e o trabalho pesado por nós.

+

VS Code

(Visual Studio Code) - O editor que usaremos para configurar as chaves do agente.

Para preparar o terreno, precisamos baixar apenas duas ferramentas. A primeira é o Antigravity 2.0, um assistente autônomo que age como um desenvolvedor particular, fazendo o trabalho pesado por você. A segunda é o Visual Studio Code, ou VS Code.

Por que o VS Code?

"Mas eu não sei programar! Pra que um editor de código?"

  • Na prática, o VS Code é apenas um Bloco de Notas com superpoderes.
  • Nós vamos usá-lo apenas para abrir um arquivo seguro chamado .env.
  • É lá que nós colaremos a nossa Chave de API, para que fique escondida e segura no seu computador.
Muitas pessoas olham pro VS Code e entram em pânico achando que terão que programar em Matrix. Mas na verdade, para leigos, o VS Code funciona como um Bloco de Notas mais chique e seguro. Nós baixamos ele apenas porque o bloco de notas normal do Windows as vezes estraga arquivos de configuração. Nós vamos abrí-lo, clicar no arquivo .env (que é um cofre do agente), colar um texto lá e fechar. Simples assim.

Perdendo o Medo da Tela Preta

O que é um Terminal?

O terminal nada mais é do que um chat diretamente com o seu computador. Em vez de usar o mouse para clicar em pastas, você "fala" com ele digitando ordens.

Para que vamos usá-lo hoje?

  • Para configurar a nossa API do OpenRouter.
  • Para acordar (ativar) o nosso agente Cronos de Oportunidades.
Outro pavor dos iniciantes é o Terminal, aquela famosa 'tela preta'. Para desmistificar: o terminal é apenas um chat do WhatsApp, só que em vez de falar com uma pessoa, você fala com o núcleo do seu computador. E como você fala? Escrevendo comandos. Nós vamos usar a tela preta apenas para duas coisas simples: rodar a configuração inicial e para dar o botão de 'Play' no agente. E mesmo isso, faremos com ajuda.

O Primeiro Passo: Instalação Facilitada

Deixe o Antigravity 2.0 instalar o Hermes para você com um simples pedido:

"Instale o Hermes Agent no meu computador e configure o ambiente para mim."

(O Antigravity cria as pastas, baixa o orquestrador e valida a instalação de forma 100% autônoma, sem você digitar nenhum código complexo)

Para instalar o agente de verdade, você não precisa decorar um passo a passo enorme. Lembra do Antigravity 2.0? Nós vamos abrir o chat com ele e pedir em português: 'Instale o Hermes na minha máquina'. Ele vai assumir o controle (com a sua permissão) e fazer o download, instalar arquivos nas pastas corretas e configurar tudo.

Como o Antigravity Trabalha?

O que a IA está fazendo nos bastidores da sua máquina:

Terminal

O agente executa comandos no seu terminal para baixar dependências e criar pastas.

Leitura de Arquivos

Ele analisa o seu computador para saber onde organizar o sistema.

Edição de Código

O próprio Antigravity edita configurações iniciais para que tudo fique compatível.

E enquanto o Antigravity trabalha, o que ele está fazendo exatamente? Ele está usando as próprias ferramentas que descrevemos na aula anterior. Ele usa a ferramenta Terminal para clonar repositórios, a de Leitura para ver se a pasta já existe, e a de Edição para preparar os arquivos-base. É uma IA configurando outra IA.

Passo 1: A Chave no OpenRouter

A Alma do Seu Agente

A chave de API é o "passaporte" que conecta o seu Hermes aos super cérebros como ChatGPT, Llama 3 ou Claude.

  1. Acesse o site openrouter.ai
  2. Crie uma conta gratuita
  3. Vá em "Keys" e clique em Create Key
  4. Copie o código gigante (ex: sk-or-v1-xxxx...)
Assim que a instalação termina, o agente existe, mas é como um cérebro desligado. A inteligência vem da Chave de API, que conecta a estrutura local à capacidade de raciocínio de ponta. Acessem openrouter.ai, criem a chave e deixem no CTRL+C. O OpenRouter é excelente porque nos dá acesso a dezenas de modelos de IA num único lugar, pagando apenas centavos por uso.
